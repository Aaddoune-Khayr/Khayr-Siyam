### **Exemple d'un Hadith détaillé :**

```
┌─────────────────────────────────────┐
│ 📖 Les jours blancs (13, 14, 15)   │
├─────────────────────────────────────┤
│                                     │
│ 💡 Explication:                     │
│ ┌─────────────────────────────────┐ │
│ │ Les jours blancs correspondent │ │
│ │ aux 13e, 14e et 15e jours de   │ │
│ │ chaque mois lunaire. Ces jours │ │
│ │ sont appelés "blancs" car la   │ │
│ │ lune est pleine et éclaire la  │ │
│ │ nuit. Le jeûne de ces trois    │ │
│ │ jours équivaut au jeûne d'un   │ │
│ │ mois entier.                    │ │
│ └─────────────────────────────────┘ │
│                                     │
│ 📜 Hadith:                         │
│ ┌─────────────────────────────────┐ │
│ │ D'après 'Abdel Malik Ibn       │ │
│ │ Qudama Ibn Malhan, d'après son │ │
│ │ père (qu'Allah les agrée), le  │ │
│ │ Prophète (que la prière d'Allah│ │
│ │ et Son salut soient sur lui)   │ │
│ │ nous ordonnait de jeûner les   │ │
│ │ jours blancs, le 13, le 14 et │ │
│ │ le 15 et il disait: « Ceci est │ │
│ │ comme le jeûne du mois entier ».│ │
│ │                                 │ │
│ │ (Rapporté par Nasai et         │ │
│ │ authentifié par Cheikh Albani  │ │
│ │ dans Sahih Taghrib n°1039)     │ │
│ └─────────────────────────────────┘ │
│                                     │
└─────────────────────────────────────┘
```

## 🚀 **Pour tester l'application :**

1. **Installez Flutter** : [flutter.dev](https://flutter.dev/docs/get-started/install/windows)
2. **Ouvrez un terminal** dans le dossier `Application`
3. **Exécutez** : `flutter pub get`
4. **Lancez** : `flutter run`
